COMPATIBLE
--------------------------------------------------------------------------------------------------------------
WildGrass from ejhopkins - http://bit.ly/WildGrass
This texture pack also works with this mod. All textures are included already in the terrain.png. 

FEATURES
--------------------------------------------------------------------------------------------------------------
This texture pack contains several existing elements from current texture packs combined. But the main base for
this pack is the BEST texture pack for minecraft in my opinion and that is John Smith texture pack V6 together 
with less contrasted terrain.png

Here a full list of used texture packs:

JohnSmith [32x32] V6/7 - http://bit.ly/JohnSmithTexturePack
V6 - Main base for my theme is the less contrast alternative terrain.png and Follage/grass.png 
V7 - Bed,wool colors and plants. 
V7.1 - Grass block and items.png.

Dusty [32x32] - http://bit.ly/DustyTexturePack
1.25.11 - Mohjang logo, painting and particles.png.

Misa's Realistic [64x64] - http://bit.ly/MisaRealisticTexturePack
v1.90 - Used the amazing HD font over the normal font and mobs. Dial, shadow, vignette, water.png.

Brown & Bloom [32x32] - http://bit.ly/BrownGloomTexturepack
v1.2.02 Classic - Sponge, music and jukebox block.

PainterlyPack - http://bit.ly/PainterlyPack
The great white/grey cloud.png.

INSTALL
--------------------------------------------------------------------------------------------------------------
NOTE! ONLY USE THE HD TEXTURE FIX TO INSTALL THIS PACK.

1.Download the latest version of xau's HD TEXTURE FIX http://bit.ly/HDtextureFix
2.Unzip it and then run it (Make sure Minecraft is closed).
3.Click the browse button at texture pack option and select SJSF.zip.
4.Change the Tile Size to 32*32 if isn't already.
5.At the options section everything should be checked apart from custom portal.
6.Click "Patch" button.
7.Close the patcher and open minecraft and enjoy the SJSF texturepack!
